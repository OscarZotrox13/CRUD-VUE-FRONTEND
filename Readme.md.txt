Pasos para ejecturar la aplicacion fontend


Instalar o tener instalado Node.js 
tener el npm


abrir una terminal en la carpeta

cd frontend

hay que instalar las dependencias
npm install

ejecutar el servidor de desarrollo que seria

npm run serve


acceder a la aplicacion desde el localhost

http://localhost:8080


las tecnologias utilizadas fueron

vue 3 
Vuex
Vue Router para la navegacion
vuetify
localStorage


Caracteristicas principales



CRUD COMPLETO de clientes
Frontend con Vue 3, Vuex y Vue Router
